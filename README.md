## Commands

    npx nx@latest init

    npx nx serve

    npx nx build

When run again you see this:
    
    Nx read the output from the cache instead of running the command for 1 out of 1 tasks.

Caching with tests
    
    npx nx test --watch=false

    npx nx graph
